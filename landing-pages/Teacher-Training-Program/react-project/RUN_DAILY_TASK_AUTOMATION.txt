DAILY AUTOMATIC TASK ALLOCATION - RUN STEPS

1. Extract this ZIP.
2. Open PowerShell inside the extracted project folder.
3. Run backend:

cd backend
npm install
npm run dev

4. Open another PowerShell in the main project folder and run frontend:

npm install
npm run dev

5. Open the frontend URL shown by Vite, usually:

http://localhost:5173

6. Check backend health:

http://localhost:5000/health

7. Test daily automation APIs with Postman or Thunder Client:

Generate 20 dummy automation teachers:
POST http://localhost:5000/api/daily-task-automation/teachers/generate-dummy

Import included FinalT2.xlsx sample activity dataset:
POST http://localhost:5000/api/daily-task-automation/activities/import-sample

Generate today's daily task allocation:
POST http://localhost:5000/api/daily-task-automation/generate-daily
Body:
{
  "activityCount": 4,
  "replaceExisting": true
}

Check today's assignments:
GET http://localhost:5000/api/daily-task-automation/today

Check teacher T001 dashboard tasks:
GET http://localhost:5000/api/daily-task-automation/teacher/T001/today

Check teacher T001 notifications:
GET http://localhost:5000/api/daily-task-automation/teacher/T001/notifications

Note:
- The activity count is flexible. Change DAILY_ACTIVITY_COUNT in backend/.env or send activityCount in API body.
- Daily automatic schedule is controlled by DAILY_TASK_CRON in backend/.env.
- Default schedule is Monday-Friday at 7:30 AM.
- Make sure MongoDB connection in backend/.env is correct before running backend.
